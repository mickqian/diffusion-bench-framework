Where the gains come from

Lossless inference-stack gains

Preserve the generation recipe. Improve throughput, latency, and memory by changing how the serving stack schedules, places, parallelizes, and executes the pipeline.

  • Dynamic batching — keep the GPU busy
  • Full-stack layerwise offload — fit larger models
  • VAE parallel decode — split across GPUs
  • Model-aware parallelism — TP / SP / CFG
  • Disaggregation — Mooncake, split stages
  • Shared CUDA kernel stack — reuse tuned kernels
  • Breakable CUDA graph — runtime kernel selection
Lossy acceleration frontier

Change the amount or precision of computation. Report speed only together with quality drift, artifact checks, and an explicit acceptance threshold.

  • Quantization — FP8 / FP4 / NVFP4
  • Temporal cache — reuse across steps
  • Progressive resolution — low-res, then upscale
  • Attention backends — swappable for speed

Built to extend

Beyond raw speed, SGLang-Diffusion is an open, modular serving platform — built on Diffusers, composable by stage, and observable — so research and industry can build on it, not just run it.

Diffusers-native backend

Loads Hugging Face Diffusers pipelines directly, so new models work day-0 — and you can freely compose Cache-DiT, parallelism (TP / SP / CFG), quantization, and LoRA on top.

Modular pipeline

Every model is a composition of stages — text-encode → denoise → decode. New algorithms like progressive resolution slot in as stages without forking the engine.

Profiling & observability

A built-in profiler with per-stage and per-step timing makes denoise and decode bottlenecks visible, so tuning is measured rather than guessed.

Broad model coverage

20+ image and video families — Wan, FLUX, HunyuanVideo, LTX, Qwen-Image, Cosmos and more — plus 3D generation and high-speed inference for real-time world models, all served on one stack.
